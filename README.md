# ANC-Net
ANC-Net论文的核心源码。The kernel source of the ANC-Net.


## 论文主要创新点 The key contributions of the paper
1. 搭建了一种多尺度特征提取器。 Employ a multi-scale feature extractor.
2. 提出了一种离散小波变换池化层。 Employ a discrete wavelet transform (DWT) pooling layers.
